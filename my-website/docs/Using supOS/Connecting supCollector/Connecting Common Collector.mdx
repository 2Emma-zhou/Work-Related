## Authenticating Collector
Use the UUID generated on the supOS platform, authenticate and connect supOS with the collector.

1. Log in to supCollector, and then select **RTD Collector > Connection Info**.<br />
One collector can upload data to up to 5 supOS platforms. There is a default supOS platform on the list under **Connection Info**.
2. Click **Add**, and then enter information of the collector and the supOS platform on which the collector is created.

          Log in to the supOS platform, and then click <img

7.png"
          /> at the upper-right corner to go to the design center.

<li className="stepexpand">
        <span>
          Select{" "}
          <span className="menucascade">
            <strong>Collection Node Management</strong> > <strong>Status</strong>
          </span>
          .
        </span>
      </li>
      <li className="stepexpand">
        <span>
          Select the collector you just added, view the details and then
          click <strong>OK</strong>.
        </span>
        <div>
          <ul><li>The collector is connected, and a new instance under <b>Object Model Management > Object Instance > Entity Template > Collector Template</b> is created.</li>
          <li>An <b>Online</b> property with data type as <b>Bool</b> is generated under the instance, and you can use it in APP designing to indicate collector status (true means online and false is offline).
          </li>
          </ul>
:::info
- Video collector that is added to supOS platform generates a new instance under <b>Object Model Management > Object Instance > Entity Template > Video Template</b>.
- Alarm collector that is added to supOS platform generates a new instance under <b>Object Model Management > Object Instance > Entity Template > AE Template</b>.
        </div>
      </li>
    </ol>
