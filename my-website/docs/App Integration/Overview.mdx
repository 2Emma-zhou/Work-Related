Complete industrial applications can be integrated to supOS based on corresponding integration rules.

<table>
<caption>Integration types</caption>
<tr>
<th>Type</th>
<th>Description</th>
</tr>
<tr>
<td>Light integration</td>
<td>Integrates the application by Iframe, using an URL with login information to avoid second login inside supOS.</td>
</tr>
<tr>
<td>Surface integration</td>
<td>With SSO enabled, industrial applications can be integrated to supOS through simply an URL.</td>
</tr>
<tr>
<td>Deep integration</td>
<td>

- Hosted: Integrates applications with supOS that is deployed on local server, and application functions, UI can be changed to adapt supOS UI.
- SaaS: Integrates applications with supOS that is deployed on cloud, and make them available on Bluetron app store.

</td>
</tr>
</table>
