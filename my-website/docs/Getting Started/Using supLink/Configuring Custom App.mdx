### Configuring on web

1. Log in to supLink web, and then click **App Manage**.
2. Under **Custom App**, click **New**.
![](https://wordpressfreezonex.oss-accelerate.aliyuncs.com/supLink/50.png)

<table>
<thead>
<tr>
<th>Parameter</th>
<th>Description</th>
</tr>
</thead>
<tbody>
<tr>
<td>Home url</td>
<td>The URL you are directed to when tapping the app on mobile.</td>
</tr>
<tr>
<td>Management url</td>
<td>The URL that directs to the management backend of the app. You can access by tapping <b>Config</b> on the app on mobile.</td>
</tr>
<tr>
<td>List layout</td>
<td>Select the layout of displaying multiple pages and folders.</td>
</tr>
<tr>
<td>Visible range</td>
<td>Set the application visible range. Select roles to grant permissions when set <b>Visible range</b> to <b>Part visible</b> or <b>Part invisible</b>.</td>
</tr>
<tr>
<td>Select group</td>
<td>Select a app group for the application.</td>
</tr>
</tbody>
</table>

3. Click **Save**.

### Viewing Effect on Mobile

1. Install supLink mobile on your phone, start it and enter the supLink server address, tap **Start**.
2. Log in with the same account as supOS.
3. Under **Custom App**, tap the app, and you will be directed to the URL you set on web.
